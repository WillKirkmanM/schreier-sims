<h1 align="center">Schreier Sims</h1>

<p align="center">The Schreier-Sims Algorithm is a Fundamental Method in Computational Group Theory. It is used to Find the Order of a Finite Permutation Group and Provides a Structured Way to test for Membership Within that Group</p>
